Italy - EDI PEC (FatturaPA)

Odoo 18 EDI PEC

This module provides PEC support for Italian e-invoicing (FatturaPA) on top of the EDI stack. It allows sending invoices to SdI via PEC, receiving SdI notifications, and importing incoming supplier invoices received via PEC.

Features

  • Send e-invoices via PEC using configured SMTP server
  • Fetch PEC emails (IMAP/POP) and parse SdI notifications
  • Route PEC messages, update EDI states on invoices
  • Import incoming e-invoices from PEC attachments
  • Company configuration of PEC servers and usage flag

Configuration

  • Enable the PEC channel in Company settings
  • Mark ir.mail_server as PEC SMTP and set credentials
  • Mark fetchmail.server as PEC and activate the cron

Dependencies

  • Odoo: mail, fetchmail
  • Odoo/OCA: l10n_it_edi, l10n_it_edi_extension

Maintainers

Odoo Community Association (OCA) and contributors.

Descrizione (Italiano)

Il modulo aggiunge il supporto PEC (Posta Elettronica Certificata) per la Fatturazione Elettronica italiana (FatturaPA) su Odoo 18, integrandosi con lo stack EDI. Consente l'invio delle fatture via PEC verso SdI, la ricezione delle notifiche SdI via PEC e l'importazione delle fatture passive ricevute via PEC.

Caratteristiche

  • Invio fatture elettroniche via PEC con il server SMTP configurato
  • Ricezione email PEC (IMAP/POP) e parsing delle notifiche SdI
  • Instradamento dei messaggi PEC e aggiornamento degli stati EDI sulle fatture
  • Import delle fatture passive da allegati PEC
  • Configurazione aziendale di server PEC e attivazione canale

Configurazione

  1. In Impostazioni > Aziende > Configurazione PEC Fatture, attiva "Usa PEC per Fatture".
  2. In ir.mail_server, configura le credenziali PEC e spunta "E-invoice PEC SMTP".
  3. In fetchmail.server, configura la casella PEC (IMAP/POP) e spunta "E-invoice PEC server".
  4. Verifica che il cron "Fetch E-invoice PEC Emails" sia attivo.

Uso

  • Dalla fattura attiva, usa l'invio EDI: se la PEC Ã¨ abilitata, l'invio avviene via PEC.
  • Le notifiche SdI in arrivo dalla casella PEC sono applicate alla fattura corrispondente.
  • Le fatture passive ricevute via PEC vengono importate e collegate come allegati.
